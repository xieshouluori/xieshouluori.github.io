我们先来看看箭头函数的语法：

一、param 是参数，根据参数个数不同，分这几种情况：
() => { … } // 零个参数用 () 表示；
x => { … } // 一个参数可以省略 ()；
(x, y) => { … } // 多参数不能省略 ()；

二、expression：

多行语句需要用{}括起来，

单行表达式不需要{}，并且会作为函数返回值

三、和普通函数一样，箭头函数也可以使用剩余参数和默认参数。
