箭头函数

语法

我们先来看看箭头函数的语法：

一、param 是参数，根据参数个数不同，分这几种情况：
() => { … } // 零个参数用 () 表示；
x => { … } // 一个参数可以省略 ()；
(x, y) => { … } // 多参数不能省略 ()；

二、expression：

多行语句需要用{}括起来，

单行表达式不需要{}，并且会作为函数返回值

三、和普通函数一样，箭头函数也可以使用剩余参数和默认参数。

特性

一、箭头函数内部没有constructor方法，也没有prototype，所以不支持new操作

二、箭头函数的 this 始终指向函数定义时的 this，而非执行时；
