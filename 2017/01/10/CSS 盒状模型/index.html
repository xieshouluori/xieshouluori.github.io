CSS 盒状模型

CSS盒子模型就是在网页设计中经常用到的CSS技术所使用的一种思维模型。

虽然 CSS 中没有盒状模型这个属性，但它却是 CSS 中无处不在的。

CSS盒子模型有：外边距（margin）、边框（border）、内边距（padding）、内容（content）四个属性。

css盒模型分为两种

一种是遵循w3c标准的标准盒模型，另一种是IE盒模型。
两种盒子模型都包括padding，margin，border，content四部分；
但是ie盒子模型的content已经计算border和padding在内。
