weex / weex-toolkit 脚手架(mac)

安装 node

注意： node 版本要大于 5.0

检查 Node的当前版本

node -v

管理Node版本

即使用Node Binary管理模块“n”
安装n模块

安装node最新的稳定版本

sudo n stable

安装weex-toolkit

$ npm install -g weex-toolkit
