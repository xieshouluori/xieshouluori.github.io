babel 6.0

介绍

Babel 是一个通用的多用途 JavaScript 编译器。通过 Babel 你可以使用（并创建）下一代的 JavaScript，以及下一代的 JavaScript 工具。

用最新标准编写的 JavaScript 代码向下编译成可以在今天随处可用的版本。 这一过程叫做“源码到源码”编译， 也被称为转换编译。

babel-node

如果你要用 node CLI 来运行代码，那么整合 Babel 最简单的方式就是使用 babel-node CLI，它是 node CLI 的替代品。

它支持Node的REPL环境的所有功能，而且可以直接运行ES6代码。
它不用单独安装，而是随babel-cli一起安装。然后，执行babel-node就进入PEPL环境。

babel-polyfill

Babel默认只转换新的JavaScript句法（syntax），而不转换新的API，比如Iterator、Generator、Set、Maps、Proxy、Reflect、Symbol、Promise等全局对象，以及一些定义在全局对象上的方法（比如Object.assign）都不会转码。Babel默认不转码的API非常多。

JavaScript 模块化编程

一 模块模式

（1） 闭包与 IIFE

模块模式的本质:
(1)可以定义私有变量和函数，外部无法访问它们，从而做到了私有成员的隐藏和隔离。

(2)通过返回对象或函数,可以公开我们对外暴露的公开的方法与数据。

（2）参数输入

如果模块内需要使用其它全局对象，应该将这些对象作为参数来显式引用它们

将引用的对象作为参数，使它们得以和函数内的其它局部变量区分开来

（3）模块输出（Module Export）

通过匿名函数的 return 语句来声明和输出模块中的对象

（4）与其它模块规范或 JS 库的适配

AMD 定义了 define 函数，可以使用 define 来编写模块化的代码
CommonJS定义了exports 或是 module.exports

以上代码可以让我们的代码既兼容于 CommonJS Modules 或 AMD 规范，又能在一般环境下运行

style、currentStyle、getComputedStyle的区别

css设置样式分为三种方式

1.内嵌样式（inline Style） ：写在Tag里面的style属性中，只对所在Tag有效。

2.内部样式（internal Style Sheet）：是写在HTML的style标签内部，只对所在的网页有效。

3.外部样式表（External Style Sheet）：一个以.css为后缀的CSS文件，然后在每个需要用到这些样式(Styles)的网页里引用这个CSS文件。

js获取样式

1.style属性：

通过document.getElementById(id).style.XXX就可以获取到XXX的值。(可读可写)

作用：只能取到通过内嵌方式设置的样式值，即style属性里面设置的值

2.runtimeStyle ：

通过document.getElementById(id).runtimeStyle.XXX设置

作用：运行时的样式！如果与style的属性重叠，将覆盖style的属性！

3.currentStyle 与 getComputedStyle

（1）作用：获取元素中所有可用的css属性

（2）应用：IE使用currentStyle，其他浏览器用getComputedStyle

（3）语法：

1> getComputedStyle：(只读不可写－－－只能获取样式，不能设置)

获取属性列表

arr_style=window.getComputedStyle(元素,伪类)

伪元素，是否要获取伪元素属性值。如hover,active,link等属性。如果不想获取这些伪元素的属性值请填写为null。

eg1: style = window.getComputedStyle(elem_id , ":after");

eg2：var width = window.getComputedStyle(mydiv , null);

获取具体的属性值：

eg1：var width = window.getComputedStyle(mydiv , null)['width'];

eg2：var width = window.getComputedStyle(mydiv , null).width;

eg3：var width = window.getComputedStyle(mydiv , null).getPropertyValue("width");

获取float，如果使用键值访问，则不能直接使用getComputedStyle(element, null).float，而应该是cssFloat与styleFloat，自然需要浏览器判断了，比较折腾！

2> currentStyle

eg:var width = mydiv.currentStyle[‘width’];
